
🔐 Day 6/100 – Password Attacks & Credential Security

Today I learned how hackers steal passwords using different attack techniques like phishing, brute force, credential stuffing, and password spraying.

I also learned why weak passwords are considered a major vulnerability in cybersecurity and how SOC Analysts monitor suspicious login attempts to detect attacks before accounts get compromised.

Key Takeaways:
• Weak passwords = easy target
• Phishing tricks humans, not systems
• Failed login monitoring is important
• User awareness is critical in cybersecurity
